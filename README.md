# 아이유 응원법 초성 리듬게임

리듬게임으로 즐기는 아이유 응원법

<https://iu-fanchants.web.app/>

### Prerequisites

- [Yarn Berry](https://github.com/yarnpkg/berry)

### 실행

```console
$ yarn workspace iu-fanchants-web dev
```

### Export licenses

```console
$ yarn licenses generate-disclaimer -R > packages/web/public/licenses.txt
```
